# prismpreview.github.io

Public documentation site for Prism until a custom domain is ready.

This folder is designed to become the root of the public GitHub Pages repository:

```text
prismpreview/prismpreview.github.io
```

It is static HTML/CSS/JS and does not require a build step.

## Publish Later

After the Prism organization and GitHub Pages repository exist, copy the contents of this folder into that repository and enable GitHub Pages from the default branch.
